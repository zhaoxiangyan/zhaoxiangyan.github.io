# 数据类型

值类型（基础类型）： 字符串（String）、数字（Number）、布尔（Boolean）、空（Null）、未定义（Undefined）、Symbol。

引用数据类型（对象类型）： 对象（Object）、数组（Array）、函数（Function），还有两个特殊对象：正则（RegExp）、日期（Date）。

注： Symbol 是 ES6 引入了一种新的原始数据类型，表示独一无二的值。
